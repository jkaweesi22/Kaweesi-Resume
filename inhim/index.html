What this project is about

The web is loud. Your new tab does not have to be. In Him is a small, intentional surface: truth first, beauty second, distraction never. Scripture is loaded locally; backgrounds come from Unsplash with a graceful gradient when images are unavailable.

Every time you open a tab, you get one more chance to pause—and read.

Features

Everything you need for a focused daily habit—nothing you don’t.

KJV scripture corpus

Verses ship with the extension from a local scriptures.json file—readable offline for the text itself.

Time-of-day blocks

Morning, afternoon, and evening each keep their own scripture and background—persisted so your rhythm feels consistent.

Backgrounds & fallback

Unsplash-powered imagery with a tasteful gradient when the network or image fails—never a broken empty tab.

Favorites

Save verses you return to often—overlay, quick select, and remove when you are done.

Section filter

Browse all scripture sections or narrow to the one you are studying—without leaving the new tab.

Manifest V3

Built for modern Chrome: a proper new-tab override with today’s extension platform.

Keyboard shortcuts

Fast hands, calm screen.

N next scripture B next background F favorites Esc close overlay

Privacy, by design

No accounts. No analytics. Scripture stays local; only background images may load from Unsplash when you are online—nothing tracked for advertising.
